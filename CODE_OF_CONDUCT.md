# Code of Conduct

This project is for focused, respectful technical work.

Expected behavior:

- be direct without being hostile
- argue from evidence, reproduction steps, and concrete tradeoffs
- keep issues and pull requests on topic

Unacceptable behavior includes harassment, personal attacks, threats, sustained
derailing, or publishing private information without permission.

Maintainers may edit or delete comments, close or lock threads, block accounts,
or take other reasonable action to keep the project usable.

Use GitHub's reporting and moderation tools for conduct problems that should
not be handled in a public project thread.
